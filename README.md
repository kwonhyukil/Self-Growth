## 프로젝트명: 自己成長ログ (Self Growth Log API)

---

# 1. 프로젝트 개요

## 1.1 목적

- 사용자가 사소하게 기뻤던 순간을 기록한다.
- 한국어로 상황과 칭찬 문장을 먼저 작성한다.
- 동일한 내용을 일본어로 직접 작문한다.
- 작성한 일본어 문장을 자동 문법 검사로 검증받는다.
- 피드백을 통해 일본어 작문 능력을 향상시킨다.

본 프로젝트는 **기록 서비스 + 일본어 학습 루프 시스템**이다.

---

# 2. 기술 스택

## 2.1 Backend

- Node.js
- Express
- TypeScript (권장)
- Prisma ORM
- MySQL
- JWT 인증
- LanguageTool API 연동
- Swagger (API 문서화)
